# AceIQ Website

State-of-the-art marketing website for AceIQ — the AI-powered study companion for South African CAPS & IEB students.

## Built With
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Features
- 🎨 Glassmorphism design with warm academic colors
- ✨ Smooth scroll animations and hover effects
- 📱 Fully responsive (mobile-first)
- 💰 Pricing section with NO SUBSCRIPTIONS emphasis
- 🤝 Affiliate program section
- 🇿🇦 South African focused (Rands, CAPS/IEB, Afrikaans)

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

The build output will be in the `dist/` folder, ready for GitHub Pages or Vercel.

## Deployment

### GitHub Pages
1. Push this repo to GitHub
2. Go to Settings > Pages
3. Select "Deploy from a branch" and choose `main` / `dist`

### Vercel
1. Push to GitHub
2. Import project on Vercel
3. Deploy automatically
